MY LEARNING PATH

一条从基础到
项目的学习路径

采用“先手写、再用框架重构”的方式。每个阶段都要有可运行的产出，而不是只收藏资料。

  1. 01Python 与 API写工具函数、FastAPI 路由、异步请求和最小 LLM 调用。
  2. 02Agent 核心理解提示词、Tool Calling、ReAct 循环与短期/长期记忆。
  3. 03RAG 工程完成文档处理、检索、重排序、评估与可追溯回答。
  4. 04状态化工作流用 LangGraph 重构流程，再进入多 Agent 协作与综合项目。
