给 Agent 开发者的 Python 基础清单

学习 Python 不需要先刷完所有语法。对 Agent 项目而言，先能读写数据、组织函数、处理错误和调用 API 更重要。

学习笔记 · 依据本地《Phase 0 Day 1：Python 基础速通》整理

为什么从数据结构开始

Agent 的工具定义、对话消息、模型返回和检索元数据大多是 JSON 形状的数据。在 Python 里，这对应列表和字典。先熟悉遍历、追加、查找和嵌套访问，才能读懂工具调用结果，也能把自己的结果整理成稳定的接口输出。

六类能力与实际用途

能力在 Agent 中的用处
变量与基础类型保存配置、状态和模型输出。
列表与字典组织 tools、messages、JSON 结果和元数据。
条件与循环根据模型返回决定下一步，遍历检索结果。
函数把工具调用、解析和格式化拆成可测试单元。
文件读写加载资料、保存日志、输出中间产物。
异常处理面对网络、模型和输入失败时安全降级。

一个适合练习的小任务

写一个读取 Markdown 文件的脚本：输出文件名、标题数量和前几段内容；遇到路径不存在或编码异常时给出明确提示。这个任务同时练到函数、字典、文件 I/O 和异常处理，也会直接成为后续 RAG 文档加载器的雏形。
