PYTHON · FOUNDATION

给 Agent 开发者的 Python 基础清单

学习 Python 不需要先刷完所有语法。对 Agent 项目而言，先能读写数据、组织函数、处理错误和调用 API 更重要。

学习笔记 · 依据本地《Phase 0 Day 1：Python 基础速通》整理

先掌握六类能力

能力在 Agent 中的用处
变量与基础类型保存配置、状态和模型输出。
列表与字典组织 tools、messages、JSON 结果和元数据。
条件与循环根据模型返回决定下一步，遍历检索结果。
函数把工具调用、解析和格式化拆成可测试单元。
文件读写加载资料、保存日志、输出中间产物。
异常处理面对网络、模型和输入失败时安全降级。

第一阶段目标不是写“大项目”，而是独立写出几十行可运行的脚本。能把输入、处理、输出和错误分开，后面接 API 才不会失控。
