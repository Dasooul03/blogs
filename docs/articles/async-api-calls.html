PYTHON · ASYNC

异步不是更快，而是别空等

调用模型、检索服务或外部 API 时，程序大部分时间都在等待网络。异步的价值是把等待时间交还给其他任务，而不是让单个请求神奇地变快。

学习笔记 · 依据本地《Phase 0 Day 3：异步编程 + 调外部 API》整理

三个基本规则

规则含义
async def声明这个函数会进行可等待的操作。
await等待一个异步结果，同时把执行权交回事件循环。
异步客户端用支持 async 的 HTTP 客户端发请求，避免阻塞整个服务。

在 Agent 服务中的使用

并行获取工具结果、调用 Embedding、轮询任务状态都是适合异步的地方。但如果函数内部仍在做大量同步计算或阻塞 I/O，外层写成 async 也不会带来收益。
